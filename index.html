Frequently Asked Questions

What is a DNS Changer?
DNS (Domain Name System) is like the phonebook of the internet. MZR DNS changes your device's DNS server to a faster, more secure one, bypassing your ISP's default slow or restricted servers.
Is MZR DNS safe to use?
Absolutely. MZR DNS uses Android's native VPN connection framework (locally) to change DNS settings without Root. We do not decrypt your traffic, and we do not store any logs.
Does it require Root access?
No! MZR DNS works on all Android devices without Root access. It creates a local loopback interface to route DNS queries safely.
Is it really free?
Yes, MZR DNS is 100% free to use. We believe privacy is a right, not a luxury. There are no hidden subscriptions.
